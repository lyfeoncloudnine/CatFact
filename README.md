# CatFact

CatFact app made with Combine.

### Features

* Using Only UIKit, Combine.
* Implement Reactor Concept via Combine.
* No OpenSources Libraries used.

### Requirements

* Xcode 14.3.0+
* iOS 15.0+

### How to install

* Just download, Build and Run.
